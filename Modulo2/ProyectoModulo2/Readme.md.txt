Sistema de Gestión de Contactos

Este proyecto es una aplicación en Python que permite gestionar una agenda de contactos desde la consola, utilizando programación orientada a objetos, listas y diccionarios.

Funcionalidades

Agregar nuevos contactos

Editar contactos existentes

Eliminar contactos

Buscar contactos por nombre o teléfono

Mostrar la lista de contactos


Requisitos

Python 3

No requiere librerías externas


Cómo ejecutar el programa

Clonar o descargar este repositorio.

Abrir una terminal en la carpeta del proyecto.

Ejecutar el siguiente comando:

python contactos.py


Tecnologías usadas

Python

Programación Orientada a Objetos

Listas y diccionarios
